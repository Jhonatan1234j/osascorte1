Reflexão e Aplicação Prática

Propriedades CSS utilizadas:

  • display: flex; — transforma um container em flexível
  • justify-content — alinha itens no eixo principal (horizontal)
  • align-items — alinha itens no eixo transversal (vertical)
  • flex-direction — define a direção dos itens
  • space-around / space-between / center — definem diferentes tipos de espaçamento entre os itens

Aplicação prática com Flexbox:

Se eu fosse fazer um site com cards responsivos, usaria Flexbox para alinhar os cards lado a lado. Com display: flex e flex-wrap: wrap, os cards se adaptariam ao tamanho da tela. Usaria justify-content: space-evenly para distribuir os cards igualmente e align-items: center para alinhar verticalmente. Isso deixaria o layout limpo e adaptável para celular ou computador.
